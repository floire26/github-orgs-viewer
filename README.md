# github-orgs-viewer
Project to show github organisations (in development)

To run :
1. Make sure that you have Node.js v.18+ installed
2. Run this code from the root
    ```
    cd client && npm i && npm run dev
    ```

To test:
1. Make sure that you have Node.js v.18+ installed
2. Run this code from the root
    ```
    npm i && npm run test
    ```
